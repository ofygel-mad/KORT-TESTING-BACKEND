## 1. Архитектурные решения

### Навигация (расширяемая на весь KORT)
Вся навигация Чапана вынесена в массив `NAV`. Когда Чапан будет встраиваться в главный сайдбар KORT — достаточно переключить `chapan-nav` на общий компонент. Ничего не захардкожено.

```js
const NAV = [
  { id: 'orders',     label: 'Заказы'    },
  { id: 'production', label: 'Цех'       },
  { id: 'shipping',   label: 'Отправка'  },
  { id: 'warehouse',  label: 'Склад'     }, // ← полноценный пункт
  { id: 'invoices',   label: 'Накладные' },
  { id: 'purchase',   label: 'Закуп'     },
  { id: 'analytics',  label: 'Аналитика' },
];
```

### Иерархия данных (Макеты 1–4)
```
▶ Абай бомбер · 24 пол.
  ▶ по размерам · итого 24
      › 42 = 10   › 44 = 8   › 46 = 6
  ▶ по цветам · итого 24
      › Белый = 12   › Чёрный = 8
  ▶ по SKU · позиций 3
      [цвет | размер | длина | наличие | резерв | доступно | статус]
```

### Топбар — все контролы в одной строке
```
[Заголовок "Склад"]
[Поиск] [Фильтр ▾] [Вид ▾] | [Показать метрики] [+ Добавить] [Экспорт]
```

**Вид** dropdown — два раздела:
- **Отображение:** Каталог (дерево) / по SKU (плоская таблица)
- **Плотность:** Стандартный / Компактный

**Метрики** — скрыты по умолчанию, без информационного шума.

## 6. Review: стоил ли рефакторинг?

### Вердикт: Да, однозначно

**Минусы старого дизайна:**

- **Информационный хаос** — 8–10 секций на одном скролле без иерархии. Глаз не знал куда смотреть
- **Смешанные паттерны** — формы создания сущностей жили рядом с таблицами данных
- **Нет структуры данных** — нельзя было быстро посмотреть "сколько Абай бомберов размера 44 белого цвета"
- **Дубликаты** — один функционал доступен из нескольких точек, путаница
- **Технический вид** — UI выглядел как dev-tool, а не рабочий инструмент оператора склада
- **Яркие, кричащие цвета** — много синего, зелёного, красного без системы
- **Каталог вне навигации** — пользователь не знал о его существовании

**Что появилось:**

- **Чёткая точка входа** — Склад в сайдбаре Чапана, наравне с другими разделами
- **Иерархия** — товар → подгруппы → SKU, три уровня без шума
- **Calm enterprise стиль** — тёплые нейтральные тона, тихие акценты, точная сетка
- **Все контролы в одном месте** — поиск, фильтр, вид, метрики — одна строка
- **Метрики по требованию** — не загружают экран по умолчанию
- **Масштабируемая архитектура** — легко перенести на весь KORT без переписывания

**Что можно улучшить в следующей итерации:**

- Модалки для быстрых действий прямо из строки SKU (принять, зарезервировать)
- Мобильная версия — сейчас сайдбар скрывается, нужен bottom nav
- Реальный live feed в виде тонкого индикатора в топбаре
- Drag & drop сортировки в каталоге
